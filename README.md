This is a chatbot example that will help customers find the right product and place an order via the chatbot.
You will need the following environment variables to run the example locally:

#### Pinecone
```
PINECONE_INDEX_NAME

PINECONE_API_KEY

PINECONE_INDEX_HOST
```

#### Azure OpenAI
```
AZURE_OPENAI_API_KEY

AZURE_OPENAI_MODEL

AZURE_OPENAI_RESOURCE_NAME
```


#### Azure Embedding
```
AZURE_EMBEDDING_API_KEY

AZURE_EMBEDDING_ENDPOINT

AZURE_EMBEDDING_API_VERSION
```


#### Contentstack
_If you want to pull data from Contentstack_
```
CONTENTSTACK_CMS_STACK_API_KEY

CONTENTSTACK_CMS_DELIVERY_TOKEN

CONTENTSTACK_ENVIRONMENT

CONTENTSTACK_CMS_REGION

CONTENTSTACK_CMS_BRANCH

CONTENTSTACK_CMS_CONTENTTYPE
```


#### Shopify
```
SHOPIFY_DOMAIN

SHOPIFY_STOREFRONT_ACCESS_TOKEN

SHOPIFY_ADMIN_ACCESS_TOKEN
```
